Hellow World - GitHub Guides

10分で読めます

あなたは以下のようなことを学べます。

  • リポジトリの作成と使用
  • 新しいブランチの開始と管理
  • ファイルを変更して、コミットとしてGitHubへpushする
  • pullリクエストの公開とマージ
